Warum IMHWS?

Für Handwerker gemacht,
nicht für Bürokraten

IMHWS wurde speziell für die Bedürfnisse von Handwerkern, Kleinunternehmen und Freiberuflern entwickelt. Kein over-engineered ERP-System, sondern genau das was Sie brauchen.

  • Desktop-App — Ihre Daten bleiben auf Ihrem Computer
  • Keine monatlichen Kosten, kein Abo, kein Abo-Falle
  • Funktioniert komplett offline — kein Internet nötig
  • SQLite-Datenbank — robust, schnell, portabel
  • Windows, macOS (Intel + Apple Silicon)
  • Export nach PDF, DATEV, CSV, ICS, SEPA, XRechnung
  • Import aus DATANORM, BMEcat, GAEB, CSV
  • Automatische Updates — immer aktuell
  • Monteur-Rapporte syncen & direkt in Rechnungen importieren
  • Sicherheit: SQL-Injection-Schutz, XSS-Sanitization, Server-Hardening
  • GitHub Actions CI/CD — automatische Builds für alle Plattformen
