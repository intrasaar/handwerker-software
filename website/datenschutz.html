3. Datenerfassung in dieser Software

3.1 Lokale Datenspeicherung

Die Software speichert sämtliche eingegebenen Daten (Kunden, Rechnungen, Angebote, Buchungen etc.) ausschließlich lokal auf Ihrem Computer in einer SQLite-Datenbank. Es werden keine Daten an externe Server übertragen.

3.2 Kein Tracking und keine Telemetrie

IMHWS erfasst keine Nutzungsdaten, kein Verhalten, keine Analysen und keine Diagnosedaten. Es werden keine Cookies gesetzt und keine Tracking-Pixel verwendet.

3.3 E-Mail-Versand

Wenn Sie die integrierte E-Mail-Funktion nutzen, um Rechnungen oder Angebote an Ihre Kunden zu versenden, werden die E-Mail-Adressen Ihrer Kunden lokal verarbeitet und über Ihren konfigurierten SMTP-Server versendet. Der Anbieter hat keinen Zugriff auf diese E-Mails oder deren Inhalte.

3.4 Lizenzverwaltung

Bei der Aktivierung der Software wird eine anonymisierte Hardware-Kennung erhoben, um die Lizenz an Ihren Computer zu binden. Es werden keine personenbezogenen Daten in Zusammenhang mit der Aktivierung gespeichert. Die Lizenzüberprüfung erfolgt lokal ohne Internetverbindung.
